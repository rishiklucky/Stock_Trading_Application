# Stock Trading Application

A polished full-stack stock trading platform built to demonstrate real-world web application architecture and trading workflow.

## What this project is

This application simulates a stock trading experience with both backend services and a modern browser interface. It is designed to show how a trading platform can support user accounts, live portfolio tracking, stock selection, and transaction processing.

## What it does

- Allows users to register and log in securely
- Displays available stocks and supports symbol-based search
- Enables users to buy and sell stocks through a trading interface
- Tracks portfolio holdings and displays current positions
- Shows transaction history for trading activity
- Provides charts for portfolio distribution and stock performance

## Why it matters

This project demonstrates several important software engineering skills:

- Building a REST API with Java and Spring Boot
- Managing relational data with Spring Data JPA and MySQL
- Creating a responsive frontend with React and client-side routing
- Integrating charting libraries for data visualization
- Designing a clear separation between frontend and backend concerns

## Architecture

The repository is organized as a two-part application:

- `stock-app-backend/` – Spring Boot REST API that handles authentication, stock data, trading logic, and portfolio management
- `stock-app-frontend/` – React application that provides the user interface, chart visualizations, and API integration

## Key technologies

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- React
- Axios
- Chart.js / react-chartjs-2
- Springdoc OpenAPI / Swagger UI
