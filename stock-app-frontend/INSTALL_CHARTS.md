# Installing Chart Dependencies

To use the interactive charts feature, you need to install the required dependencies.

## Installation Steps

1. Navigate to the stock-app directory:
```bash
cd stock-app
```

2. Install the chart libraries:
```bash
npm install chart.js react-chartjs-2
```

3. Start the development server:
```bash
npm start
```

## What's Included

The Charts page includes:
- **Stock Price Comparison** - Bar chart comparing current prices vs previous close
- **Stock Price Trend** - Line chart showing high vs low prices
- **Portfolio Distribution** - Pie chart showing quantity distribution
- **Portfolio Value Distribution** - Doughnut chart showing value distribution

## Access the Charts

Once installed, navigate to: `http://localhost:3000/charts`

Or click the "Charts" link in the navigation bar.
